# Voice Assistant Skill

This skill tunes AnyClaw for voice-like interaction patterns.

## Focus

- Short, speakable responses.
- One actionable instruction at a time.
- Clear confirmations before risky steps.
- Honest fallback to text when no direct speech tool is available.

## Notes

- This skill is prompt-only today.
- It improves conversational behavior but does not yet synthesize or record
  audio by itself.
- It pairs well with `app-controller` and `vision-agent` for hands-free style
  desktop guidance.
